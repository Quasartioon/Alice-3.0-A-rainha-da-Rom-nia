# Alice 3.0 A rainha da Romênia
Projeto com código para bot no Discord para fins de diversão e entretenimento

# Pacotes e Tecnologias necessárias
O código utiliza Node.js é necessário que instale o npm e o proprio node para continuar

Os pacotes são:
discord.js, google genai, axios, dotenv e fs (nativo do Node.js)

para instalar:
npm install discord.js
npm install @google/genai
npm install axios
npm install fs
